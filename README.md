# storybook-sample

storybookのサンプル

## Project Setup
npmでもok
nodeのバージョン管理はvoltaを推奨

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
pnpm test:unit
```

### Run Storybook

```sh
pnpm storybook
```

### Run Storybook Tests

```sh
pnpm storybook:test
```

### Run Storybook Screenshot

```sh
pnpm storybook:screenshot
```
